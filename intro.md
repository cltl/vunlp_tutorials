# NLP Transformers - Tutorial Notebooks

These tutorial notebooks are part of the NLP course in the [Master AI](https://vu.nl/en/education/master/artificial-intelligence) at the Vrije Universiteit Amsterdam, taught by dr. [Lucia Donatelli](https://research.vu.nl/en/persons/lucia-donatelli/). The notebooks are created by [Leon Eshuijs](https://watermeleon.github.io/).

The goal of these labs is to walk you through the transformer architecture with hands-on coding examples, from tokenization and attention, to building a transformer from scratch, to working with pretrained models. Each notebook can be run directly in Google Colab (via the button at the top of each page) or locally. For local setup instructions, see the readme of the [GitHub repository](https://github.com/cltl/nlp_transformer_tutorials). 

### Labs
- [Lab 1 - Transformer Basics](nlp_transformer_tutorials/NLP_Lab1_Transformer_Basics): tokenization, text generation, and attention
- [Lab 2 - Building Transformers](nlp_transformer_tutorials/NLP_Lab2_Building_Transformers): implementing a transformer from scratch
- [Lab 3 - Pretrained Transformers](nlp_transformer_tutorials/NLP_Lab3_Pretrained_Transformers): generation, fine-tuning BERT for classification, and the logit lens
